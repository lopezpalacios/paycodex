# Glossary

- **ADDACS** — Bacs DDI cancellation / amendment message
- **AEOI** — Automatic Exchange of Information (tax)
- **AFP** — Association for Financial Professionals (US)
- **AISP** — Account Information Service Provider (PSD2). See [[regulations/psd2-psd3]]
- **AMLA** — Anti-Money Laundering Authority (EU body, Frankfurt). See [[regulations/amld-amlr-amla]]
- **AMLD** — Anti-Money Laundering Directive (EU). See [[regulations/amld-amlr-amla]]
- **AMLR** — Anti-Money Laundering Regulation (EU). See [[regulations/amld-amlr-amla]]
- **AMLO** — Anti-Money Laundering Ordinance (CH/FINMA). See [[regulations/swiss-amla]]
- **AP** — Accounts Payable
- **API** — Application Programming Interface. See [[concepts/api]]
- **APP fraud** — Authorised Push Payment fraud (UK reimbursement scheme)
- **AR** — Accounts Receivable
- **ARUDD** — Bacs unpaid return message
- **ASF** — Available Stable Funding ([[concepts/nsfr]])
- **AUDDIS** — Bacs automated DDI advice
- **AWACS** — Bacs account amendment message
