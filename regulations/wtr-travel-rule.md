# WTR / Travel Rule (Regulation EU 2023/1113)

Wire Transfer Regulation recast. Includes crypto-asset transfer rules.

## Core requirement

Information on payer + payee must "travel" with the payment. Required fields:

- Payer name, account / IBAN, address (or BIC)
- Payee name, account / IBAN
- Both readable in payment data, retained by both PSPs

## Recast 2023 — key changes

- Extends to crypto-asset service providers (CASPs) — MiCA tie-in
- Removes thresholds for some flows (full info required even sub-€1k)
- Aligns with FATF Recommendation 16

## In CH

- FATF-aligned via Swiss AMLA + AMLO-FINMA
- Travel rule applies to CH↔EU and CH↔crypto
