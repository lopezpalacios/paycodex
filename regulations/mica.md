# MiCA — Markets in Crypto-Assets Regulation (EU 2023/1114)

EU crypto regulatory framework. Phased application: stablecoins June 2024, broader Jan 2025.

## Scope

- Crypto-Asset Service Providers (CASPs)
- Issuers of crypto-assets (asset-referenced tokens / e-money tokens / other)
- Excluded: NFTs (mostly), CBDC, security tokens (under MiFID)

## Cash-mgmt link

- Banks providing crypto custody / payment services need CASP authorization
- [[wtr-travel-rule]] extension to crypto via 2023/1113
- Stablecoin issuance + reserves rules
- KYC + AML at level of crypto-fiat onramps

## CH context

- CH not in scope, but Swiss CASPs targeting EU clients need authorization
- CH FinTech license + DLT Act provides domestic equivalent
- Swiss banks (UBS, ZKB, SEBA, Sygnum) early movers on regulated crypto
