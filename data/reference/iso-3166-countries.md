# ISO 3166 countries

Country code list. ISO 3166-1 alpha-2 used in payment messages.

Examples relevant: AT, BE, CH, DE, ES, FR, GB, IT, LU, NL, PL, PT, SE, US, etc.

## Validation

- Mandatory in CBPR+ structured address (post-Nov 2025 cross-border)
- Must be active code (TF, AN, YU all retired)
- Cross-check vs sanctions country list

## Subdivision codes (ISO 3166-2)

- Used for state/province in some address formats
- e.g., CH-ZH (Zurich), DE-BY (Bavaria)
