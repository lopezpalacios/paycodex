# Purpose codes (ExternalPurpose1Code)

ISO 20022 code list. Identifies underlying reason for payment. Used in `<Purp><Cd>`.

## Common subset

| Code | Meaning |
|---|---|
| SALA | Salary payment |
| PENS | Pension |
| GOVT | Government payment |
| TAXS | Tax payment |
| SUPP | Supplier payment |
| TRAD | Trade settlement |
| INTC | Intra-company |
| LOAR | Loan repayment |
| DIVI | Dividend |
| INSU | Insurance premium |
| RENT | Rent |
| UTIL | Utilities |
| CHAR | Charity |
| BENE | Unemployment benefit |
| CBLK | Card bulk clearing |

Full list: ISO 20022 ExternalCodeSets, published by ISO 20022 RA. Hundreds of codes.

## Why fill it

- Regulatory reporting (esp cross-border)
- AML / fraud signal
- Beneficiary categorization
- Sanctions compliance
