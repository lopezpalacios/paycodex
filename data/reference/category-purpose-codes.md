# Category purpose codes (ExternalCategoryPurpose1Code)

Higher-level grouping than [[purpose-codes]]. Used in `<CtgyPurp><Cd>`.

## Examples

| Code | Meaning |
|---|---|
| CASH | General cash management |
| CORT | Trade settlement |
| DIVI | Dividend |
| INTC | Intra-company |
| LOAN | Loan |
| PENS | Pension |
| SALA | Salary |
| TAXS | Tax |
| TREA | Treasury |
| TRAD | Trade |
| INTE | Interest |
| SECU | Securities |

## Why both purpose + category-purpose

- Purpose is fine-grained
- Category-purpose drives processing path (intra-company gets different treatment than salary)
- Some banks use category-purpose to route to different sanction rule sets
