# Lockbox

PO Box owned by bank. Bank opens mail, deposits checks, sends remittance data to corp.

## Variants

- **Wholesale** — B2B, low volume, high \$, manual processing
- **Retail** — consumer payments, high volume, scannable coupons, OCR

## Output

- Image archive of checks + remittance docs
- Data feed (BAI2, custom file) to AR system
- Daily deposit report

## Status

Declining product — replaced by [[ach]], [[rtp-fednow]], [[virtual-accounts]]. Still alive in industries with paper-heavy AR (insurance, healthcare, gov).
