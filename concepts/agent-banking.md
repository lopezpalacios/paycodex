# Agent banking

Non-bank entity (retailer, post office, mobile operator) providing limited banking services on behalf of licensed bank.

## Examples

- Swiss Post providing PostFinance services in branches
- M-Pesa agent network (Kenya, similar models elsewhere)
- Convenience stores cashing cheques + accepting deposits (US)
- Travelex providing FX over the counter

## Vs BaaS

- Agent banking = physical / counter service
- BaaS = digital / API-led
- Both: non-bank channel for regulated bank

## Reg

- Local regulation varies — agent permitted or not
- Typically AML responsibility stays with bank
- Limits per agent (cash thresholds, transaction caps)
