# STET API

French Open Banking API standard. Used by major French banks under PSD2.

## Origin

- Developed by STET (French CSM operator + technical body)
- Adopted by BNP Paribas, Société Générale, Crédit Agricole, BPCE, others

## Coexistence with Berlin Group

- STET is dominant in France
- Some French banks expose Berlin Group additionally (e.g., for cross-border TPPs)
- TPPs targeting France typically integrate both

## Structure

Similar shape to Berlin Group:

- AIS endpoints
- PIS endpoints
- CBPII funds confirmation
- SCA flow with redirect / decoupled
