# Berlin Group NextGenPSD2

Pan-European Open Banking API standard. Most-adopted bank-to-TPP API spec in EU.

## Scope

- AIS (account info)
- PIS (payment initiation)
- CBPII funds confirmation
- Standardized data model + endpoints + auth

## Adopted by

- Most German + Austrian banks
- Many across CEE + Benelux + Nordics
- France: STET API used instead (some banks support both)
- Spain + Italy: regional standards plus Berlin Group

## Versions

- v1.3 widely deployed
- v1.3.x updates ongoing
- Implementation support via aspsp-config endpoint

## Vs UK Open Banking standard

- UK uses Open Banking Implementation Entity (OBIE) standard — different shape
- Berlin Group is EU-focused, OBIE is UK-only
