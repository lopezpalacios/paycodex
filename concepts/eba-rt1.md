# EBA RT1

Private real-time pan-EU instant payment system. Run by EBA Clearing.

## Properties

- Live since Nov 2017 (predates [[tips]] go-live for SCT Inst)
- 24/7, ≤10s, EUR
- ISO 20022 (pacs.008 inst)
- Bank-owned, opt-in participation

## Vs [[tips]]

- RT1 = private, older
- TIPS = ECB-operated, becoming dominant under IPR
- Interoperability via TIPS bridge
