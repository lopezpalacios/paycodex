# FX Spot

Foreign exchange transaction settling typically T+2 (some pairs T+1, USDCAD T+1, RUB T+0).

## Use

- Cover known short-term obligations
- Convert receipts to home currency
- Speculative + market-making

## Quote

- Bid / Ask spread
- Mid-market rate
- Spread = bank's primary revenue + provision

## Settlement

- T+2 standard (e.g., trade Mon, settle Wed)
- Settlement via [[cls]] for CLS-eligible pairs (most G10)
- Non-CLS: bilateral via correspondent chain

## Cash mgmt link

- Treasury executes spot for known funding gaps
- Pricing transparency: best execution under MiFID II if applicable
- Drives nostro position changes ([[nostro-vostro]])
