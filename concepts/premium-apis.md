# Premium APIs

Bank-published APIs beyond regulated PSD2 minimum. Monetized.

## Examples

- **Real-time balance** (PSD2 covers this for AIS, but premium adds richer data)
- **Multi-bank statement aggregation** (bank as aggregator host)
- **Payment status with rich metadata** (UETR tracking, fraud reasoning)
- **FX rate + execute** APIs
- **Trade finance APIs** (LC / BG status, document upload)
- **Cards APIs** (issuance, virtual cards)
- **Treasury services** (sweep config, sub-account creation)

## Monetization

- Per-call pricing
- Tiered subscription
- Volume bands
- Some included free with relationship value

## Standards

- Banks publish own OpenAPI specs
- BIAN (Banking Industry Architecture Network) reference models
- Each bank's developer portal

## Adoption

- Growing — corp clients increasingly API-first
- TMS vendors integrate via premium APIs
- Differentiator for cash mgmt sales
