# FATF — Financial Action Task Force

Intergovernmental AML / CFT standard-setter. 40 Recommendations.

## Mandate

- Sets global AML / CFT standards
- Mutual evaluations of member jurisdictions
- Greylist / blacklist of non-compliant countries

## Why it matters

- All major jurisdictions implement FATC standards via national law
- WTR / Travel Rule = FATF Recommendation 16 implementation
- Greylist countries trigger EDD by global banks
